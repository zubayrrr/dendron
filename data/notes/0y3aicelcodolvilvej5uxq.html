BIOS

Newer version of BIOS is UEFI

BIOS is the firmware on the Motherboard, firmware instruction set of a motherboard, it can be upgraded, it is burned into the chip of the motherboard.

We can configure things such as:

  • Power on password
  • Driver encryption
  • Secure boot(that makes sure OS hasn't been tampered with) Secure boot is specific to UEFI
