hard link

  • Areas: Linux
  • It is an association between an inode structure and a file name in a directory.
  • You cannot create a hard link to a directory.
  • All hard links have some inode structure and inode number

Example

  • ps aux > processes.txt
    • To create a hard link (Private): ln processes.txt p.txt
    • If we were to change the location of processes.txt, p.txt will not be effected.
