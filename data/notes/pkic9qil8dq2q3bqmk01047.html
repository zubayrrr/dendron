Hub

A Hub is an older technology that connects network devices together (such as Client and Server).

A hub is a physical layer networking device which is used to connect multiple devices in a network. They are generally used to connect computers in a LAN. Creating a Daisy-Chaining effect.

Hubs can lead to increased network errors.

A hub received information in one port and rebroadcasts it out all the other ports.

  • Layer 1 device used to connect multiple network devices/workstations
  • Known as multiport repeaters
  • Three basic type of Ethernet Hubs:
    • Passive Hub
      • Repeats signal without any amplification
    • Active Hub
      • Repeats signal with amplification
    • Smart Hub
      • Active Hub with enhanced features like SNMP
