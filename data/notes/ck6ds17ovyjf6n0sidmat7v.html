Networking in Linux

  • Areas: Linux, Networking
  • Getting Information about the Network Interfaces
    • ifconfig (Private), ip
    • lo stands for loopback (inteface)
    • enp0s3 (Private) for the Ethernet interface
    • There are other names for interfaces such as wlo1 (Private) , wlan0 (Private) for wireless interface.
  • When configuring network intefaces using ip, ipconfig (Private), route you must run them as root(sudo)
  • Network inteface configuration set by ifconfig (Private) or ip are not persistent, after a system restart all the changes are lost. To make them persist, we need to edit distro specifc configuration file. For example on Ubuntu you have to use netplan, systemd-networkd (Private) for static configuration of the network.
