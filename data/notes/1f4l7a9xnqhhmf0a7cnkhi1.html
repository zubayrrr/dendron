groupadd

  • Areas: Linux

groupadd command is used to create a new user groups (Private).

Syntax:

sudo groupadd [option] group_name

  • sudo groupadd engineering create a new group
  • tail -3 /etc/group or cat /etc/group - check last 3 groups or all groups inside /etc/group
  • Adding users to the group can be acheived using useradd (Private), note that users can only be added to groups that have already been created using groupadd
  • groupmod (Private) can be used to make changes to the group name, groupdel (Private) to delete a group
